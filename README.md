# mesopotamie

Site Vue 3 + Vite sur la mythologie mesopotamienne.

## Developpement local

```bash
npm install
npm run dev
```

## Build de production

```bash
npm run build
npm run preview
```

## Deploiement GitHub Pages

Le deploiement est automatique via GitHub Actions sur chaque push de la branche `main`.

Configuration requise dans GitHub:

1. Ouvrir `Settings > Pages` du depot.
2. Dans `Build and deployment`, choisir `Source: GitHub Actions`.

URL finale attendue:

- https://jd-orkest.github.io/mesopotamie/
